# ContactManagerApp

A Swift-based iOS application designed to manage personal and professional contacts efficiently.

## Features

- **Add New Contacts**: Input details such as profile picture, name, surname, phone number, notes, email, address, and birthdate.
- **Edit Existing Contacts**: Update contact information seamlessly.
- **Delete Contacts**: Remove contacts from the list as needed.
- **Image Support**: Add or update contact photos using the device's photo library.
- **Persistent Storage**: Save and retrieve contacts using Core Data for data persistence.
- **Dark Mode Support**: Automatically adapts to the user's system settings to provide a seamless experience in both light and dark themes.

## Technologies Used
- **Language**: Swift
- **Frameworks**: SwiftUI, UIKit
- **Data Storage**: Core Data
- **Architecture**: MVVM (Model-View-ViewModel)

## Project Structure
- **Models**: Defines the data structures.
- **ViewModels**: Handles business logic and data manipulation.
- **Views**: Contains SwiftUI views for the user interface.
- **Helpers**: Utility functions and extensions to support the app's functionality.

## Getting Started

### Prerequisites
- **Xcode**: Version 12.0 or higher
- **iOS**: Deployment target of iOS 14.0 or higher
- **macOS**: Version 10.15 or higher

### Steps
1. Clone the Repository:
   ```bash
   git clone https://github.com/emirokay/ContactManagerApp.git
2. Navigate to the Project Directory:
	```bash
	cd ContactManagerApp
3. Open in Xcode:
	```bash
	open ContactManagerApp.xcodeproj
4. Build and Run:
- Select a simulator or connect a physical device.
- Click the 'Run' button in Xcode to build and launch the app.
